Analysis

Series:

Step 1: Is it a known series?

  • Geometric series : ⅀qk
    • |q| < 1 ⇒ Absolutly convergent
    • Else divergent
  • Harmonic series : ⅀1/ks
    • s > 1 konvergent
    • s ≤ 1 divergent
Step 2: Is ak a null sequence? for k → ∞ = 0
  • No? → Divergent
  • Yes? Go to step 3!
Step 3: Convergence criteria
  • Quotient criterion |ak+1/ak|
    • < 1 absolutly konvergent
    • > 1 divergent
    • = 1 no result
  • Leibniz criteria
    1. Alternating series
    2. |ak| monotonous zero sequence? → conditionally convergent
    3. If ⅀|ak| also is convergent → Seires is absolutly convergent
  • Comparison criteria
    • ak ≤ bk and bk is konvergent → ak is also konvergent
    • ak ≥ bk and bk is divergent → ak is also divergent

L'Hospital rule:

Step 1: Use value (limit) if it leads to 0 * 0, −+∞ * −+∞, 0/0 or −+∞/−+∞ → derive and check (change product into fraction!) else continue step 2!

Step 2: Use value and calculate result

Derivatives:

  • General derivation: n * xn-1
  • (x) → cos(x) → −sin(x) → −cos(x)
  • ln(x) → 1/x → −1/x2 → −2/x3
  • e−x → −e−x → e−x
  • √x → 1/2√x
